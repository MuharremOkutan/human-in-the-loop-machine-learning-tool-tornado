<h2>To launch Tornado</h2>
<code>$ sh ./launch_tornado.sh</code>
<p>Tornado will then be available at http://0.0.0.0:3000</p>
  
<h2>To stop Tornado</h2>
<code>$ sh ./stop_tornado.sh</code>
